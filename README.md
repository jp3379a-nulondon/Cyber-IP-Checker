<summary>Introduction</summary>

### Purpose

The purpose of this project is to design and develop a cyber security web application that allows users to enter an IP address and check its reputation from a single interface.

The business requirement is to provide the cyber security team with a centralised source for IP reputation checking, reducing the need to manually visit multiple external tools or websites. The application aims to improve efficiency during security investigations by allowing analysts to submit an IP address and view relevant reputation information in one place.
